# Package index

## All functions

- [`test.clusters.MC()`](https://gonzalez-delgado.github.io/PCIdep/reference/test.clusters.MC.md)
  : Test for the difference of two cluster means after any clustering
  algorithm, for matrix normal model with arbitrary scale matrices.
- [`test.clusters.hc()`](https://gonzalez-delgado.github.io/PCIdep/reference/test.clusters.hc.md)
  : Test for the difference of two cluster means after hierarchical
  clustering, for matrix normal model with arbitrary scale matrices.
  Supported linkages (as in Gao et al. 2022) are "single", "average",
  "centroid", "ward.D", "median", "mcquitty" and "complete".
- [`test.clusters.km()`](https://gonzalez-delgado.github.io/PCIdep/reference/test.clusters.km.md)
  : Test for the difference of two cluster means after k-means
  clustering, for matrix normal model with arbitrary scale matrices.
